## IndentGuide Plugin for Eclipse

### Description

Adds configurable indent guide lines in Eclipse text editors.

### Preferences

The preference page can be found at 

> **`Window`&rarr;`General`&rarr;`Editors`&rarr;`Text Editors`&rarr;`Indent Guide`**
